## üîí Security & Trust Features

### 1. **Secure Payment System with Security Deposits**
- **What it does:** Renters pay rental amount + security deposit upfront
- **How it protects you:**
  - Deposit is held securely until rental completion
  - Full refund if item returned in good condition
  - Automatic deductions only for verified damage
  - Clear breakdown of all charges

### 2. **OTP Verification System**
- **Handover OTP:** Secure verification when item is picked up
- **Return OTP:** Secure verification when item is returned
- **Why it matters:**
  - Prevents fraud and unauthorized pickups
  - Creates accountability trail
  - Both parties must verify each transaction
  - Auto-expires for security

### 3. **Condition Verification with Photos**
- **At Handover:**
  - Take photos of item condition
  - Rate condition (1-5 stars)
  - Add notes about any existing scratches/marks
- **At Return:**
  - New photos for comparison
  - Rate current condition
  - Automatic damage detection by comparing ratings
- **Benefits:**
  - Visual proof of item condition
  - Fair damage assessment
  - Protects both renter and owner

### 4. **Automatic Damage & Late Fee Calculation**
- **Damage Charges:**
  - System compares condition ratings
  - Significant drops trigger damage assessment
  - Charges deducted from security deposit only
  - Dispute resolution if disagreement
- **Late Fees:**
  - Automatic calculation if returned late
  - 50% penalty on daily rental rate
  - Clear upfront in rental terms
- **Transparency:** All charges calculated and shown before deduction

### 5. **Insurance Options**
- **Optional Coverage:** Choose insurance for high-value items
- **What's Covered:**
  - Accidental damage
  - Lost items
  - Theft protection
- **Low Premium:** Small percentage of rental amount
- **Peace of Mind:** Both parties protected

## üöÄ How to Rent Safely

### For Renters:

1. **Browse & Search**
   - Use advanced filters to find items
   - Check owner's ratings and reviews
   - View all item details and photos

2. **Book the Item**
   - Select rental dates
   - Option to add insurance
   - Pay rental amount + security deposit
   - Money held securely

3. **Handover (Pickup)**
   - Meet owner or arrange delivery
   - Owner provides OTP code
   - Verify and enter OTP in app
   - Take photos of item condition
   - Rate condition (important!)
   - Add notes about any existing marks

4. **Use & Enjoy**
   - Use item as agreed
   - Take care of it
   - Contact owner via in-app chat if issues

5. **Return**
   - Return on or before end date
   - Owner verifies with return OTP
   - Take photos of current condition
   - Owner checks condition

6. **Get Refund**
   - If condition maintained: Full deposit refunded
   - If minor wear: Agreed deduction
   - If damage: Fair assessment and deposit adjustment
   - Refund processed automatically

### For Owners:

1. **Create Listing**
   - Upload clear photos
   - Detailed description
   - Set rental price per day
   - Set security deposit amount
   - Set min/max rental days

2. **Review Rental Requests**
   - Check renter's profile and ratings
   - Accept or decline
   - Generate handover OTP when accepted

3. **Handover**
   - Meet renter or arrange pickup
   - Share handover OTP
   - Wait for renter to verify and document condition
   - Keep copy of condition photos

4. **During Rental**
   - Be available for questions via chat
   - Track rental period

5. **Return Verification**
   - Meet renter or arrange return
   - Use return OTP to verify
   - Check item condition thoroughly
   - Take photos to compare with handover photos

6. **Complete & Get Paid**
   - If condition good: Full deposit refunded to renter
   - If damage: Fair amount deducted and added to your payment
   - Rental payment released to you
   - Leave review for renter
